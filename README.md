# Sanctuary

A browser extension for blocking websites and managing your digital wellness.

## Features

- Block distracting websites
- Manage blocked sites easily
- Customize blocklist settings
- View your browsing activity

## Installation

1. Clone this repository
2. Open your browser's extension management page
3. Enable "Developer mode"
4. Click "Load unpacked" and select the project directory

## File Structure

- `background/` - Background scripts and service workers
- `blocked/` - Blocked sites functionality
- `options/` - Settings and configuration page
- `popup/` - Extension popup interface
- `icons/` - Extension icons
- `utils/` - Utility functions
- `manifest.json` - Extension manifest
